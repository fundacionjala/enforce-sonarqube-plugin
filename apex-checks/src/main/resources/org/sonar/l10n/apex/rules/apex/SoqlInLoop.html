SOQL statements should not be in loops.

Noncompliant Code Example

public void someMethod {
	...
	while (true) {
		Contact c = new Contact(Account = [SELECT Name FROM Account WHERE NumberOfEmployees > 10 LIMIT 1]);
	}
}

Compliant Solution

public void someMethod {
	...
	Contact c = new Contact(Account = [SELECT Name FROM Account WHERE NumberOfEmployees > 10 LIMIT 1]);
}
