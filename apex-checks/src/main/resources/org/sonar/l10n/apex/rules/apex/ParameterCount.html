For better readability avoid too many parameters in methods/constructors.

Noncompliant Code Example

The maximum number of parameters in method/constructor is 5.

  void someMethod(integer arg1, integer arg2, integer arg3, integer arg4, integer arg5, integer arg6) { // violation
      }
  
  class SampleClass {
      SampleClass(integer arg1, integer arg2, integer arg3, integer arg4, integer arg5, integer arg6, integer arg7) { // violation
      }
  }

Compliant Solution

        void someMethod(integer arg1) {
        }

    class SampleClass {
        SampleClass(integer arg1) {
        }
    }
