"test.startTest()" and "test.stopTest()" methods should only be called once each inside a testMethod.

Noncompliant Code Example

@isTest
public static testMethod void someTestMethod {
    test.startTest();
    ...
    System.assert(a.equals(expected));
    test.stopTest();
    test.startTest();
    ...
    System.assert(b.notEquals(expected));
    test.stopTest();
}

Compliant Solution

@isTest
public static testMethod void someTestMethod {
    test.startTest();
    ...
    System.assert(a.equals(expected));
    System.assert(b.notEquals(expected));
    test.stopTest();
}
