Passing an assertion message that displays in case of failure makes testing easier to fix and understand.

Noncompliant Code Example

@isTest
public static testMethod void someTestMethod {
...
System.assert(a.equals(expected));
}

Compliant Solution

@isTest
public static testMethod void someTestMethod {
...
System.assert(a.equals(expected), 'The value of a is supposed to be equal to expected');
}
