Putting a DML statement inside a for loop is a performance problem since it can send more DML requests than necessary. Instead, create a list containing all of the records you want to query and include them all in a single DML statement.

Noncompliant Code Example

for(int i = 0; i < 10; i++) {
    insert newAcct; 
}

Compliant Solution

for(int i = 0; i < 10; i++) {
    ...
}
insert newAcct;
