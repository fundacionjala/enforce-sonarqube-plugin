Test methods should have at least one assertion and must have the testMethod keyword.

Noncompliant Code Example

@isTest
public class SampleTest() {
    static testMethod void someTestMethod() {
    //a block of code, without System.assert 
    }
}


@isTest
public class SampleTest() {
    static void someTestMethod() {
    //block of code.
    System.assert(something); // this is a test context.
    }
}

Compliant Solution

@isTest
public class SampleTest() {
    static testMethod void someTestMethod() {
    ...
    System.assert(something);
    }

    ...

    static testMethod void someTestMethod() {
    ...
    System.assertEquals(something, otherThing, 'message');
    }
}

public class SampleTest() {
    static testMethod void someTestMethod() {
    ...
    System.assert(something);
    }

    ...

    static testMethod void someTestMethod() {
    ...
    System.assertEquals(something, otherThing, 'message');
    }
}
